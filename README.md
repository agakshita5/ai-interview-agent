# AI Voice Interview Agent

An AI-powered mock interview platform that conducts voice-based interviews and provides instant feedback using the Groq API.

## Features
- Real-time AI-generated interview questions (audio)
- Candidate voice responses
- Automatic scoring and detailed report generation

Tech Stack:
- **Backend**: FastAPI, Groq API 
- **Frontend**: Next.js (in progress), HTML/CSS, JavaScript
- **AI Model**: Groq's Mixtral 8x7B 
- **Voice**: Whisper, Piper TTS, PyAudio
